## ‚ùå Critical SEO Gaps

### 1. **Missing Meta Descriptions**
**Impact:** HIGH ‚Äî Meta descriptions don't directly affect ranking, but they massively impact CTR from search results.

**Current:** No meta descriptions visible in source
**Issue:** Google will auto-generate snippets from content, often picking suboptimal text
**Fix:** Add custom descriptions to each post (120-155 characters, include primary keyword)

**Example for Day 1 post:**
```yaml
description: "Real data analysis of BTC funding rates on Binance: why the 3.99% APY carry trade is dead, and where the real edge might be hiding. Code + math included."
```

### 2. **No Structured Data (Schema.org)**
**Impact:** MEDIUM-HIGH ‚Äî Structured data helps Google understand content type and can enable rich results

**Missing:**
- Article schema (author, publish date, modified date)
- Breadcrumb schema
- Organization/Person schema for Ruby as author

**Benefits:**
- Potential for rich snippets in SERPs
- Better content categorization
- Author authority signals

**Implementation:** Add JSON-LD to Quarto template

### 3. **Suboptimal Title Tag Structure**
**Impact:** MEDIUM ‚Äî Titles are descriptive but not optimized for search

**Current Pattern:**
```
"Day 1: The Funding Rate Free Lunch (And Why It's Getting Stale)"
```

**Issue:** "Day X" prefix reduces keyword density and doesn't help discoverability

**Recommended Pattern:**
```
Primary Keyword | Secondary Context | Ruby's Quant Journal
"BTC Funding Rate Arbitrage: Why the Free Lunch is Gone | Ruby's Quant Journal"
```

**Rationale:**
- Front-load keywords searchers actually use
- "Day X" works for engaged readers, not for search discovery
- Keep "Day X" in H1 for existing readers, optimize title tag for search

### 4. **Missing Internal Linking Strategy**
**Impact:** MEDIUM ‚Äî No connections between related posts

**Current:** Posts are standalone
**Needed:**
- Link Day 1 funding rate post ‚Üí future contrarian signal post (Day 2)
- Link technical terms to explainer posts (when they exist)
- Create pillar content (e.g., "Ultimate Guide to Crypto Funding Rates") that links to specific deep dives

**Benefits:**
- Distributes PageRank across site
- Increases session duration (SEO signal)
- Helps Google understand topic relationships

### 5. **No Image Optimization**
**Impact:** LOW-MEDIUM ‚Äî No images currently, but will be critical when added

**When adding images:**
- Use descriptive filenames: `btc-funding-rate-chart-2026.png` not `image1.png`
- Add alt text with keywords: "BTC funding rate distribution Dec 2025 - Feb 2026"
- Compress for performance (Core Web Vitals)
- Consider WebP format

### 6. **Missing robots.txt and sitemap.xml**
**Impact:** MEDIUM ‚Äî Helps search engines crawl efficiently

**Current:** Likely auto-generated by GitHub Pages, but not verified
**Needed:**
- Explicit `sitemap.xml` with all posts
- `robots.txt` to guide crawlers
- Submit sitemap to Google Search Console
